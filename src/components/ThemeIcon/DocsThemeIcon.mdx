# ThemeIcon

## Imports

```
import { ThemeIcon, ThemeProvider } from "my-react-libary-sb";
```

### Usage
`ThemeIcon` es un componente creado para cambiar entre tema `light` y `dark` nuestra aplicación. Lo podrías colocar en cualquier lugar de tu aplicación.

<Canvas withSource="open">
  <Story id="components-themeicon--default" />
</Canvas>

Pero que este funcione no solo se necesita importar el componente, sino que también se necesita del `ThemeProvider`, el cual debe envolver por ejemplo al componente de entrada de nuestra aplicación.

<div
  style={{
    background: "rgba(0, 0, 0, 0.85)",
    padding: '10px',
    color: 'white',
    margin: '30px 0px'
  }}
>
  ```
    <ThemeProvider>
      <App />
    </ThemeProvider>
  ```
</div>

De esta manera ya el `ThemeIcon` component podrá usar de manera interna el `ThemeContext` y poder cambiar el tema de nuestra aplicación.
